Job Description: Senior ML Engineer Join the Innovation at Outnovate

Are you ready to architect intelligent systems that empower data to see, think, and act? Outnovate is seeking a Senior ML Engineer to build and deploy impactful machine learning solutions. This hybrid role based in Bangalore offers the opportunity to leverage advanced tools and collaborate across teams to deliver cutting-edge results.

Role & Responsibilities

  • Model Development: Design and implement production-ready machine learning models with Python, Spark, and TensorFlow.
  • ML Pipeline Orchestration: Harmonize ML models with cloud platforms like AWS and GCP, and MLOps tools like MLFlow and Seldon.
  • Data Processing Expertise: Handle complex data manipulations with tools like SQL, Apache Spark, and Hadoop.
  • Collaborative Integration: Partner with data scientists, engineers, and stakeholders to transform data into actionable insights.
  • Microservices Development: Develop and manage microservices using Docker and Kubernetes to ensure smooth deployment and scalability.

Mandatory Expertise

  • Time Series Analysis:
    • Develop and implement machine learning models for analyzing time series data, including trend analysis, anomaly detection, forecasting, and classification.
    • Expertise in handling large-scale, multivariate, or irregularly sampled time series datasets across domains like finance, IoT, energy, or manufacturing.
    • Perform advanced feature engineering for time series data, including the creation of lag variables, rolling statistics, and seasonal decomposition.
    • Apply statistical models such as ARIMA, SARIMA, Holt-Winters, or modern methods like Facebook Prophet for forecasting.
  • Log Data Analysis:
    • Design and develop models for extracting insights from log data, focusing on identifying patterns, anomalies, and predictive events in system/application/server logs.
    • Leverage machine learning techniques to process high-dimensional log data for use cases like predictive maintenance, cybersecurity, and root cause analysis.
    • Utilize techniques like log clustering, sequence-to-sequence models, and Hidden Markov Models (HMM) for log data mining.
  • Deep Learning & Neural Networks:
    • Design and implement deep learning architectures tailored for time series data, such as Recurrent Neural Networks (RNNs), Long Short-Term Memory (LSTMs), and Gated Recurrent Units (GRUs).
    • Experience with tree-based models like Random Forest, Gradient Boosting Machines (GBM), XGBoost, LightGBM for feature selection or predictive modeling.
    • Research and apply hybrid approaches, combining ensemble methods (e.g., Random Forest) with deep learning techniques (e.g., LSTMs) for improved predictive performance.
    • Explore advanced models like Attention Mechanisms, Transformers, and Temporal Convolutional Networks (TCNs) for time series analysis.
  • Preferred: MLOps Expertise
    • Collaborate with DevOps, data engineering, and IT teams to automate ML pipelines for end-to-end production, ensuring continuous integration and deployment (CI/CD) for ML models.

Preferred Candidate

  • Experience: 5+ years in software engineering with a focus on Python, Spark, and Java.
  • ML Expertise: Proficient in machine learning frameworks like TensorFlow and PyTorch, and libraries like scikit-learn and NumPy.
  • Cloud and MLOps Tools: Experience with AWS, GCP, and MLOps tools such as MLFlow and Seldon.
  • Full Development Lifecycle: Skilled in agile practices, RESTful API development, and task management tools like JIRA.
  • Team-Driven: Proven ability to work in cross-functional teams and collaborate on scalable ML solutions.
